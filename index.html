Proyectos

A continuación se presentan algunos proyectos desarrollados como parte de mi aprendizaje en desarrollo web, enfocados en la aplicación de buenas prácticas, optimización y estructura semántica.

Formulario de registro con Validaciones HTML

Formulario web diseñado para simular un escenario real de recopilación de datos, aplicando validaciones nativas de HTML para garantizar la integridad de la información sin necesidad de JavaScript.

Tecnologías: HTML

Lo que aprendí: Implementación de validaciones como required, pattern, min, max y control de tipos de entrada. Uso de fieldset y legend para mejorar la accesibilidad y organización del formulario.

Landing Page - Contabilidad Florez

Landing page desarrollada para presentar servicios contables, enfocada en la jerarquía visual del contenido y la navegación intuitiva para mejorar la comprensión del usuario.

Tecnologías: HTML

Lo que aprendí: Estructuración semántica del contenido, uso adecuado de encabezados y secciones, y organización de la información orientada a la conversión y experiencia del usuario.

Ver proyecto

Contenido Multimedia y Responsive

Implementación de contenido multimedia optimizado para la web, incluyendo imágenes responsivas, uso de picture para art direction y carga eficiente de video y audio.

Tecnologías: HTML

Lo que aprendí: Uso de srcset y sizes para adaptar imágenes según el dispositivo, implementación de múltiples formatos de video (MP4, WebM) para compatibilidad, y optimización de carga mediante atributos como loading y preload.

Además, se aplicaron buenas prácticas de optimización web relacionadas con peso de archivos, formatos adecuados y mejora del rendimiento, impactando métricas como tiempo de carga inicial y Largest Contentful Paint (LCP).
